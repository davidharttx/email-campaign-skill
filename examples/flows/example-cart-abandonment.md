## 4. Deployment Spec

### Template Naming

| Template Name | Flow Position | Type |
|---|---|---|
| FF \| Cart Abandonment 1 -- Cart Reminder | Step 5 | Graphic |
| FF \| Cart Abandonment 2 -- Social Proof + Urgency | Step 7 | Graphic |
| FF \| Cart Abandonment 3 -- Personal Note with Incentive | Step 10 | Text-Based |

### Klaviyo Setup Instructions

1. **Create templates** via API or Klaviyo UI using the naming convention above
2. **Create flow** in Klaviyo UI:
   - Trigger: Started Checkout OR Added to Cart metric
   - Flow filter 1: Has NOT "Placed Order" since starting this flow
   - Flow filter 2: Has NOT been in this flow in the last 14 days
3. **Build the sequence:**
   - Time Delay: 1 hour
   - Email Action: Assign "FF | Cart Abandonment 1 -- Cart Reminder" template
   - Time Delay: 23 hours
   - Email Action: Assign "FF | Cart Abandonment 2 -- Social Proof + Urgency" template
   - Time Delay: 24 hours
   - Conditional Split: "Has Placed Order at least once" (Yes -> end flow / No -> continue)
   - Email Action: Assign "FF | Cart Abandonment 3 -- Personal Note with Incentive" template
4. **Enable Smart Sending:** 16-hour window on all three emails
5. **Set flow status to Live**
6. **Create coupon in Shopify:** Static code COMEBACK10 for 10% off, no minimum, single-use per customer
