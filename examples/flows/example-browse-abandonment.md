## 4. Deployment Spec

### Template Naming

| Template Name | Flow Position | Type |
|---|---|---|
| GR \| Browse Abandonment 1 -- Product Reminder | Step 5 | Graphic |
| GR \| Browse Abandonment 2 -- Social Proof | Step 7 | Graphic |
| GR \| Browse Abandonment 3 -- Personal Note | Step 10 | Text-Based |

### Klaviyo Setup Instructions

1. **Create templates** via API or Klaviyo UI using the naming convention above
2. **Create flow** in Klaviyo UI:
   - Trigger: Viewed Product metric
   - Flow filter 1: Has NOT "Added to Cart" since starting this flow
   - Flow filter 2: Has NOT "Placed Order" since starting this flow
   - Flow filter 3: Has NOT been in this flow in the last 7 days
3. **Build the sequence:**
   - Time Delay: 1 hour
   - Email Action: Assign "GR | Browse Abandonment 1 -- Product Reminder" template
   - Time Delay: 24 hours
   - Email Action: Assign "GR | Browse Abandonment 2 -- Social Proof" template
   - Time Delay: 48 hours
   - Conditional Split: "Has Placed Order at least once" (Yes -> end flow / No -> continue)
   - Email Action: Assign "GR | Browse Abandonment 3 -- Personal Note" template
4. **Enable Smart Sending:** 16-hour window on all three emails
5. **Set flow status to Live**
