## 4. Deployment Spec

### Template Naming

| Template Name | Flow Position | Type |
|---|---|---|
| RC \| Welcome 1 -- Welcome + Discount | Step 3 | Graphic |
| RC \| Welcome 2 -- Brand Story | Step 5 | Graphic |
| RC \| Welcome 3 -- Brewing Guide | Step 8 | Graphic |
| RC \| Welcome 4 -- Personal Nudge | Step 11 | Text-Based |

### Klaviyo Setup Instructions

1. **Create templates** via API or Klaviyo UI using the naming convention above
2. **Create flow** in Klaviyo UI:
   - Trigger: Added to List (Newsletter Signup list)
   - Flow filter 1: Has NOT "Placed Order" since starting this flow
   - Flow filter 2: Has NOT been in this flow before
3. **Build the sequence:**
   - Email Action: Assign "RC | Welcome 1 -- Welcome + Discount" template (immediate)
   - Time Delay: 2 days
   - Email Action: Assign "RC | Welcome 2 -- Brand Story" template
   - Time Delay: 2 days
   - Conditional Split: "Has Placed Order at least once since starting this flow" (Yes -> end flow / No -> continue)
   - Email Action: Assign "RC | Welcome 3 -- Brewing Guide" template
   - Time Delay: 3 days
   - Conditional Split: "Has Placed Order at least once since starting this flow" (Yes -> end flow / No -> continue)
   - Email Action: Assign "RC | Welcome 4 -- Personal Nudge" template
4. **Enable Smart Sending:** 16-hour window on Emails 2-4 (Email 1 sends immediately)
5. **Set flow status to Live**
